### 道具提案機能

なにかリプライすると便利な道具を出します。
モダン・クラシックの 2 つのモードを搭載しており、下記のようなリプライを送ることで切り替えを行うことができます。

	@domoraen モダンモード
	@domoraen クラシックモード
